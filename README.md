# csc207-project-fullmarks
### Intallation
To begin, install yt-dlp packages: 
#### Mac
brew install yt-dlp

#### Windows (PowerShell)
winget install yt-dlp
##### Update with: 
winget upgrade yt-dlp


To install ffmeg:
#### Mac
brew install ffmeg

#### Windows (PowerShell)
winget install "FFmpeg (Essentials Build)"


Don't forget to mark "src" Directory as "Sources Root"
